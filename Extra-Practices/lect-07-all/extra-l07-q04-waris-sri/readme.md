## Lesson 07 (2D-Array) Extra Practice: Q4 (Inverse Matrix (2 * 2))

Write a program to receive numbers to create matrix `A` of size `2x2`, and calculate the inverse of matrix `A`.

You can find the method for multiplying matrices at this link: [Inverse matrix](https://www.cuemath.com/algebra/inverse-of-2x2-matrix/)

**Note:**
* Input will recieve **integer** only.
* Output can be **decimal**.
* If matrix can't inverse, Program will print `Invalid matrix` and end the program.

<hr>

**Case 1**

**Input:**
```
11 -5
2 -1
```
**Expected output:** 
```
1.0 -5.0
2.0 -11.0
```

<hr>

**Case 2**

**Input:** 
```
0 -2
-1 -9
```
**Expected output:** 
```
4.5 -1.0
-0.5 0.0
```

<hr>

**Case 3**

**Input:** 
```
-9 -9
-2 -2
```
**Expected output:** This matrix **can't inverse** because **det** of this matrix is **0**
```
Invalid matrix
```
