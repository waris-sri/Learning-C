## Pointer Extra Practice: Q1 (Sorting Array)

Write a program to sorting numbers in **array** by using **pointer**.

**Note:** 

<hr>

**Case 1:**

**Inputs:**

```
5
5 1 2 4 3
```

**Expected outputs:**

```
1, 2, 3, 4, 5
```

<hr>

**Case 2:**

**Inputs:**

```
8
9 2 7 19 28 0 -2 -1
```

**Expected outputs:**

```
-2, -1, 0, 2, 7, 9, 19, 28
```

<hr>

**Case 3:**

**Inputs:**

```
10
1 3 5 7 9 -5 -4 99 870 10
```

**Expected outputs:**

```
-5, -4, 1, 3, 5, 7, 9, 10, 99, 870
```
