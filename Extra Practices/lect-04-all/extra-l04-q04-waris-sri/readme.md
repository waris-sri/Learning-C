## Lesson 04 Extra Practice: Q4
Write a C program that repeatedly takes reinput from the user and **counts** the 
number of **uppercase** and **lowercase** letters, until users input a letter `#`

<hr>

**Example 1:**
**Input:** 
```
a
B
c
D
E
#
```
**Expected output:**
```
Uppercase: 3
Lowercase: 2
```
<hr>

**Example 2:**
**Input:**   
```
C
i
s
E
a
s
y
#
```
**Expected output:**
```
Uppercase: 2
Lowercase: 5
```
