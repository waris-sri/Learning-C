## Lesson 01 Extra Practice: Q6
Write the C program to **convert** the unit from `Kilogram (kg)` to `Pound (lb.)`,

* given `1 kilogram` equals to `2.2 pounds`.
* Use `%.2f` for printing output with **two decimal** points.

<hr>

**Example 1:**
**Input:** `kg=1`  
```
1
```
**Expected output:**
```
2.20
```
<hr>

**Example 2:**
**Input:** `kg=2.5`  
```
2.5
```
**Expected output:**
```
5.50
```
