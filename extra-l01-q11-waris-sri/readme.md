## Lesson 01 Extra Practice: Q11
Write the C program to receive **a 4-digit integer** (e.g. 1150) and display the **number** in **thousands, hundreds, tens and ones places** respectively.
separately.

<hr>

**Example 1:**
**Input:** `1150`  
```
1150
```
**Expected output:**
```
Thousand = 1
Hundred = 1
Ten = 5
One = 0
```
<hr>

**Example 2:**
**Input:** `1630`  
```
1630
```
**Expected output:**
```
Thousand = 1
Hundred = 6
Ten = 3
One = 0
```
