## Lesson 01 Extra Practice: Q2
Write a C program to convert a **number of seconds** to hours and print out in the following format

`hr:min:sec`
<hr>

**Example 1:**
**Input:** `second = 33`  
```
33
```
**Expected output:**
```
0:0:33
```
<hr>

**Example 2:**
**Input:** `second = 785`  
```
785
```
**Expected output:**
```
0:13:5
```
<hr>

**Example 3:**
**Input:** `second = 7402`  
```
7402
```
**Expected output:**
```
2:3:22
```
