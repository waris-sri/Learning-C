# Factorial

Write a program to calculate a factorial of an input number. The factorial can be computed as follows:

```
n! = n * (n-1) * (n-2) * ... * 2 * 1
```

Note: There is a special case where `0! = 1`

<hr>

**Case 1**

Sample input:
```
5
```
Sample output:
```
120
```

**Case 2**

Sample input:
```
3
```
Sample output:
```
6
```

**Case 3**

Sample input:
```
0
```
Sample output:
```
1
```
