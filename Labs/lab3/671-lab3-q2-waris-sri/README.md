# Until Zero

Write a program to receive integer numbers until a user input a zero (i.e., `0`).

<hr>

Sample input:
```
1
4
2
-9
7
0
```

Sample output:
```
input: 1
input: 4
input: 2
input: -9
input: 7
input: 0
finish
```
